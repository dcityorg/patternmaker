# Save & Export

## Saving Designs

**Save Design** exports all your parameters as a tiny JSON file. **Load Design** restores a saved file exactly — same seed, same settings, same pattern.

Your design is fully described by the parameters — no image data is stored, so files are typically 1–2 KB.

> ⚠️ **There is no auto-save.** If you refresh the page, unsaved changes are lost. Save frequently!

A green dot appears in the sidebar header when you have unsaved changes.

## Design Name

The design name appears below the version line in the sidebar header. Click it to rename. The name is used for Save Design and Export SVG filenames.

## Export SVG

**Export SVG** downloads a clean vector SVG file:
- No stroke (filled paths only)
- Cell and gap colors as specified
- Exact tile dimensions as set
- Ready for laser cutting, printing, or use in VaseMaker

> **Tip:** Chrome and Edge provide the best file management experience — save dialogs remember the last folder you used. Other browsers (Brave, Firefox, Safari) may not remember directories.

## The Seed

Every pattern is controlled by a **Seed number**. The same seed + same settings always produces the exact same pattern. Save your design file to preserve the seed, or note it down to reproduce the pattern later.

Click **New Pattern** to generate a new random seed variation.
